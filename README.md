# Wordle Game 🟩🟨⬛

A fully functional browser-based Wordle clone built with vanilla JavaScript — no frameworks, no dependencies.

## Live Demo
[Play Now](https://wordle-harika.netlify.app)

## Features
- 6 attempts to guess the 5-letter word
- Color-coded feedback — green, yellow, gray
- Keyboard support — type or click
- Win/loss detection with streak tracking
- Responsive design — works on mobile and desktop
- Hard mode support
- Local storage — saves your progress between sessions

## Tech Stack
- Vanilla JavaScript
- HTML5
- CSS3
- Local Storage API

## How to Play
1. Type any 5-letter word and press Enter
2. Green = correct letter, correct position
3. Yellow = correct letter, wrong position
4. Gray = letter not in the word
5. Guess the word in 6 tries to win

## Run Locally
git clone https://github.com/HarikaPedavalli/wordle-game
cd wordle-game
npm install
npm start
